# cucumber-rs-start

BDD (Behavior-Driven Development) & cucumber-rs POC.

## Cargo build on MacOS

in **~/.gitconfig** comment lines like this:

```
#[url "git@github.com:"]
#	insteadOf = https://github.com/
```

## How to run example test

```
cargo test --test example
```
